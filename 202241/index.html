从结果删除重复行

where条件查询

where只能指定行的条件，不能指定组的条件

算数运算符&比较运算符

算数运算符 + - * /

null经过运算任然是null

比较运算符

不能对null用比较运算符，而是使用 is null 运算符和 is not null 运算符。

字符串类型的数据按照字典顺序进行排序，不同于数字大小。

逻辑运算符

and，or，not

可以用括号指定优先级

写作/执行顺序

写作顺序

select-from-where-group by-having-order by

执行顺序

from-where-group by-having-select-order by

对表查询

group by

指定的列为聚合键或分组列

null也会作为一组数据

聚合键之外的别名不能在select子句出现

group by子句中不能写列的别名

where与group by同时使用

where中不能选择聚合函数

为聚合结果指定条件

having

写在group by句子之后，用来指定分组的条件。

可以设置常数、聚合函数、group by子句中指定的列名（聚合键）为条件。

对查询结果排序

order by

asc升序（null在开头），desc降序（null在末尾）

可以指定多个排序列

可以使用聚合函数和select子句中未出现的列

可以使用列编号（不建议）

insert插入

基本语法

插入多行语句用逗号将不同行隔开，全部列都插入可省略列名

写入空值和默认值

写入null就可以插入null值（not null约束的除外）

写入default就可以插入默认值（需要有default约束）

复杂查询

视图

保存的select语句，节省内存，不改变原表。

创建和删除视图

可以以视图为基础创建视图，但不建议。

视图的限制

不能用order by语句定义视图。

没有经过聚合汇总、只select了一张表、未使用distinct的视图才可以进行更新，表也随之更新。

子查询

一次性视图

基本语法

可以使用多层嵌套子查询，但不推荐

用as关键字为子查询取合适的名字

标量子查询

必须且只能返回单一值，用括号括起来使用。

能够使用常数或者列名的地方：select 子句、where 子句，group by 子句、having 子句，order by 子句，都可以使用。

where子句中使用的子查询必须是标量子查询。

关联子查询

在子查询中加入结合条件，结合条件只在子查询中生效

函数，谓词，Case表达式

函数

算数函数

abs() 绝对值

mod() 求余

round() 四舍五入

字符串函数

concat(str1, str2, ...) 拼接

upper() 或 lower() 大小写转换

length() 计算字符长度

replace(str, from_str, to_str) 替换字符串中的一部分

substr(str, start_position, char_num) 截取字符串中的一部分

日期函数

current_date 获取当前日期

current_time 获取当前时间

current_timestamp 获取当前日期和时间

extract(<日期元素> from <日期>) 截取日期元素

转换函数

cast(<值> as <想转换的类型>) 类型转换函数

coalesce(数据1, 数据2, …) 返回左边开始第一个不是null的值

谓词

返回值是真值的函数

like 一致查询

  1. % 0个字符以上的任意个字符
  2. - 1个任意字符

between 范围查询

is null/is not null 空值查询

in/not in 条件查询返回真值（可将子查询作为参数）

exist/not exist 判断是否存在某个条件的记录

  1. 通常指定关联子查询作为exist的参数
  2. 实际应用写的是 exists()
  3. exist的子查询尽量写 select *

case表达式-条件分支

相当于if，可以用在select语句中

集合运算

表的加减法

并集union

  1. 列数必须相同
  2. 列的类型必须一致
  3. order by只能用一次
  4. union all可以重复包含行

交集intersect

MySQL不可用

差集except

MySQL不可用

表的联结

通过某个列建立桥梁

on条件必须写在where子句的前面，可以join很多张表

内联结inner join

外联结outer join

使用left时from子句中写在左侧的表是主表，使用right时右侧的表是主表

交叉联结cross join

笛卡尔积
